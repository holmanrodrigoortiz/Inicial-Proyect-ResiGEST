🏠 ResiGest

Sistema de Gestión Residencial Integral

📋 Descripción del Proyecto

Este sistema web permite la gestión integral de conjuntos residenciales, incluyendo administración de propiedades, mantenimientos, residentes y todas las funcionalidades necesarias para una gestión eficiente.

🎯 Características Principales

  • Página de Login con autenticación
  • Dashboard con estadísticas en tiempo real
  • Gestión completa de propiedades
  • Control de mantenimientos y solicitudes
  • Sistema de contacto y soporte
  • Diseño responsive (móvil, tablet, desktop)

🛠️ Tecnologías Utilizadas

HTML5 CSS3 JavaScript
